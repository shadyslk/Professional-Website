Vision

The idea of this project came from the desire to showcase the projects that I have been working on. What I wanted is something simple to have as a starting point and that would be easy modify moving forward as I gain new skills.

Technologies Used

  • Amazon S3
  • Amazon Route 53
  • HTML
  • CSS

Process

I began by assessing architecture and making initial decisions. I decided to use Amazon S3's capability to host static websites. It was the best choice for my situation as it would allow me to spend less time on upkeep and it would be more cost-efficient for a starting website as I would only pay for what is actually being used.

Next, I started working on the draft of the website itself locally on my computer. I created the webpages for the three main pages: Index, About, and Portfolio. I spent plenty of time working on getting the layout and styles to fit my vision. I wanted a site that was simply styled and that had plenty of room for modifications and upgrades later down the line. I will make my code available on my github for anyone that is interested in using any of the code that I have written.

After creating the initial pages for my site, I began to work with setting up an S3 Bucket for my website to be hosted. Amazon Web Services makes this a very easy process. I was able to upload my files to this bucket and apply the configurations that I needed to host the website.

In order to make my website more easily reachable, I used Amazon Route 53 to purchase a domain name that I could use for my projects. I was able to purchase the domain name that I felt worked best for me and create a DNS record to link to my Amazon S3 hosted website. This would allow users to use an easier to remember domain name intead of the full bucket URL.

Finally, I was able to finish up some key details of my website. I spent time working on the content for the site. Amazon S3 made it very easy to upload my updated files and folder structure.

Conclusions

I have many additions and plans that I will have for the website: I am hoping to get an SSL/TLS certificate for my website to increase the overall security of data transmitted between server and client. I am also hoping to incorporate Amazon Cloudfront in the near future to improve latency in other geographical locations. That is what I have planned for the near future.
